# react-native-meituan
用react native 做的一个美团客户端首页

用的的组件：
 
* NavigatorIOS
* react-native-swiper
* react-native-icons
* react-native-navbar
* ListView
* WebView
 
### 运行步骤

```shell
  1、npm install
  2、用Xcode打开meituan.xcodeproj
  3、Commmand + R
```

界面是模仿官网的应用的
